# No More NPC
NoMoreNPC is a plugin to cancel spawning of NPC.
It was developped for Farouni from MinecraftFR Forum (https://minecraft.fr/forum/threads/reproductions-des-pnj.258307/#post-1798851).

## Configuration
No configuration needed
